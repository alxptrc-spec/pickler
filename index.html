💾 How to Save Your Data
💾 Auto-Backup (recommended)
  1. Click the 💾 Auto-Backup button in the top nav (Chrome/Edge only)
  2. Choose or create a file — ideally inside a folder synced to Google Drive, Dropbox, or OneDrive
  3. From then on, every change is silently written to that file — no clicking required
  4. Your browser may ask you to re-grant permission once per browser session — just click the button again if it says "Click to resume"
⬇ Exporting (Backing Up)
  1. Click the ⬇ Export button in the top nav
  2. A .json file will download automatically — the filename includes today's date and time
  3. Save it somewhere safe (same folder as this file, or a cloud folder like Google Drive)
  4. Export regularly — especially after a productive session
⬆ Importing (Restoring)
  1. Click the ⬆ Import button in the top nav
  2. Select your previously exported .json backup file
  3. Your progress will restore instantly
  4. Works even on a freshly downloaded version of the tracker
⚠️ When You Might Lose Progress
  • Clearing browser history or site data
  • Moving or renaming this HTML file
  • Opening the file in a different browser
  • Downloading a new version of the tracker
Always export before doing any of these.
✅ Best Practice

Keep this file in a permanent folder (e.g. Desktop → "Doc Tracker"). Export a backup every Friday. Store the backups in Google Drive or iCloud so they survive a computer wipe.
